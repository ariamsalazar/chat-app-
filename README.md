Chat APP Example

  - Chat message storage: Saves and retrieves messages via MongoDB.                                           
  - GET /api/v1/messages: Fetch messages with optional pagination (limit, before, after).
  - POST /api/v1/messages: Create a new message with author and message fields.                                
  - Real-time updates: Supports polling for new messages via the after query param.                           
  - Authentication: Protects endpoints with a Bearer token.
  - Swagger docs: Interactive API explorer at /api/v1/docs.
